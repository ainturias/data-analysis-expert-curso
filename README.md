# Data Analysis Expert

Repositorio de práctica del curso Data Analysis Expert.

Este espacio reúne materiales, ejercicios y proyectos de Excel, Power BI, SQL, R y Python organizados por módulo y semana. Los datos incluidos son de prueba y se usan con fines académicos.

## Estructura

- `Modulo 1 - Excel/`
- `Modulo 2 - Power BI Nivel 1/`
- `Modulo 3 - Power BI Nivel 2/`
- `Modulo 3 - SQL/`
- `Modulo 5 - R/`
- `Modulo 6 - Python/`

## Contenido

- Archivos de práctica y datasets de ejemplo.
- Proyectos finales y documentación de apoyo.
- Material de clase y recursos del curso.

## Nota

Algunos archivos binarios, capturas, videos y documentos de trabajo se excluyen del control de versiones para mantener el repositorio ligero y fácil de clonar.
